17. ES6+
    Q1: Explain Destructuring, Rest Operators, and Spread Operators.
    Answer:

Destructuring: Extracts individual values cleanly from arrays or object properties straight into distinct variables.

JavaScript
const { name, age } = { name: 'Alice', age: 25 };
Spread Operator (...): Unpacks and spreads elements out of an existing array or object container into a new structure.

JavaScript
const cloneArr = [...originalArr];
Rest Operator (...): Gathers multiple individual elements or dangling trailing arguments together into a clean, consolidated array container.

JavaScript
function sum(...numbers) { return numbers.reduce((a, b) => a + b, 0); }
