🔹 Rendering Architectures
Next.js supports three fundamental hybrid rendering environments depending on architectural selections:

Static Site Generation (SSG) / Static Rendering: Pages are fully rendered to static HTML at build time. Ideal for landing blogs or marketing pages.

Server-Side Rendering (SSR) / Dynamic Rendering: HTML is built fresh dynamically on the server for every single incoming request. Perfect for dynamic checkout pages or customized dashboards.

Incremental Static Regeneration (ISR): Allows you to update static pages incrementally in the background after you've built the site, without needing a full rebuild.
